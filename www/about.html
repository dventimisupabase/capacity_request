Transactional Outbox

Side effects queued in the same transaction. pg_cron sweeps, pg_net delivers. Zero lost notifications.

Auto-Expiration

Customer confirmation is time-boxed. pg_cron expires stale requests and sends reminders before deadline.

Provisioning Webhook

External systems mark requests complete or failed via authenticated API. Closes the loop automatically.

Analytics Dashboard

Requests by state, approval latency, cost by region, volume trends, time-in-state, and VP escalation rates.

Operator Guidance

Context-aware banners on every request tell operators exactly what needs to happen next.

Block Kit Messages

Rich Slack messages with status, fields, approval checks, and inline action buttons. Threaded updates.
